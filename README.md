# Iris Data Cleaning & Visualization Project

## 📊 Project Overview
This project focuses on cleaning, analyzing, and visualizing the Iris dataset using Python.

## 🔧 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📈 Key Tasks
- Data Cleaning
- Exploratory Data Analysis
- Data Visualization
- Insight Generation

## 📊 Visualizations
- Pairplot
- Heatmap
- Boxplot
- Scatter Plot

## 🔍 Key Insights
- Petal length and width are highly correlated
- Iris-setosa is clearly separable
- Minor outliers observed
- Dataset is clean and balanced

## 🚀 Conclusion
This project demonstrates data preprocessing, visualization, and storytelling using Python.
